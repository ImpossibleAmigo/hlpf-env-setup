# Практична робота №7: Redis кешування, Query параметри та Пагінація

**Виконав:** Чеберяко Олексій
**Група:** 232/1

## Опис виконаної роботи
У межах роботи реалізовано:
1.  **Пагінацію та фільтрацію** для ендпоінти товарів (`GET /api/products`) із поверненням мета-даних.
2.  **Кешування запитів** у Redis (сумісно з `cache-manager` v7).
3.  **Автоматичну інвалідацію кешу** при зміні даних.

## Результати тестування
Запит `GET /api/products?page=1&pageSize=5` повернув 31 товар (7 сторінок). Кеш у Redis успішно створюється (TTL 60s) та очищується при POST запитах.
