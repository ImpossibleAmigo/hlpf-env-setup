# Практична робота: NestJS, TypeORM та Docker

## Виконані завдання:
1. **Налаштування бази даних:**
   - Вимкнено \`synchronize\`.
   - Налаштовано TypeORM за допомогою \`DataSource\` для підтримки міграцій.
   - Створено дві міграції: ініціалізація таблиць та додавання поля \`isActive\` до продуктів.
2. **Сутності (Entities):**
   - Створено \`Category\` та \`Product\`.
   - Налаштовано зв'язок Many-to-One (багато продуктів до однієї категорії).
3. **Функціонал (CRUD):**
   - Реалізовано повний набір операцій для Категорій та Продуктів.
4. **Контейнеризація:**
   - Налаштовано \`Dockerfile\` для додатка.
   - Створено \`docker-compose.yml\`, що піднімає Node.js додаток та базу даних PostgreSQL.

## Як запустити:
1. Створити \`.env\` файл на основі \`.env.example\`.
2. Запустити команду:
   \`\`\`bash
   docker compose up --build
   \`\`\`
3. Виконати міграції:
   \`\`\`bash
   npm run migration:run
   \`\`\`
